# Vegalore: Frontend

## Directory

```bash
cd frontend
```

## Install the Frontend dependencies

```bash
npm install
```

## Run the Frontend in development mode

Standalone:

```bash
./start
```

Dockerized:

```bash
docker-compose up -d frontend
```

Then open [http://localhost:3000](http://localhost:3000) on your preferred web browser.

## Build the Frontend

```bash
npm run build
```
